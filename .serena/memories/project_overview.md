# The Bridge - Project Overview

## Purpose
An interactive narrative storytelling web application with audio-synchronized text, presenting a personal story across 12 scenes. The project combines visual storytelling, audio narration, and bilingual text (English/Urdu) to create an immersive scrolling experience.

## Core Features
- 12 narrative scenes with professional audio narration
- Synchronized text overlay with audio playback
- Bilingual support (English/Urdu)
- Keyboard navigation
- Accessibility features (screen reader support, reduced motion)
- Theme switching (dark/light mode)
- Smooth scene transitions with animations
- Audio preloading for seamless experience

## Tech Stack
- **Framework:** Next.js 16.1.1 (App Router, Turbopack)
- **React:** 19.2.3
- **TypeScript:** 5.x (strict mode enabled)
- **Styling:** Tailwind CSS v4
- **Animation:** GSAP 3.14.2, Framer Motion 12.23.26
- **Audio:** ElevenLabs API for text-to-speech generation
- **Dev Tools:** ESLint 9, tsx 4.21.0

## Project Goals
- Create an immersive, high-quality scrolling narrative experience
- Ensure full accessibility compliance
- Deliver smooth animations and audio-text synchronization
- Support both languages seamlessly
- Optimize performance with audio preloading and lazy loading
