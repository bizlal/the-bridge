## File Naming Convention
- Use kebab-case (lowercase with dashes)
- Be descriptive
- Include dates when relevant
- Examples:
  - `hamilton-spectator-2011-07-02.jpg`
  - `dailyrapfacts-dm-lil-tjay.jpg`
  - `app-store-limelight-ranking-2020.jpg`

## Supported Formats
- **Images**: .jpg, .jpeg, .png, .webp
- **Documents**: .pdf (or scanned as .jpg)
- **Videos**: ../videos/evidence/ directory

## Usage
Once files are placed here, update the scene files in `/lib/scenes/` to reference them in the `gallery` array.
